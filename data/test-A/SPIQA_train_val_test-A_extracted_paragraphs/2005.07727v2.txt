The whirlwind of progress in deep learning has produced a steady stream of promising generative modelsВ (Goodfellow etВ al., 2014; KarrasetВ al., 2018) that render natural scenes increasingly indistinguishable from reality and provide an intuitive way to generate realistic imagery given high-level user inputsВ (Wang etВ al., 2018; Bau etВ al., 2019).

We seem to be on the verge of using these generative models for semantic manipulation of natural photographs, which will combine the simplicity of pixel-level editing with flexible semantic manipulations of a scene. The role of deep generative models will be to provide latent semantic representations in which concepts can be directly manipulated and then to preserve image realism when semantic changes are made. This editing scheme will allow users to manipulate a photograph not with physical colors, but with abstract concepts such as object types and visual attributes. For example, a user can add a lit lamp into a bedroom (Figure 1 bottom left) or change the color of a treeвҖҷs leaves (Figure 1 bottom right) with a few scribbles.

Despite the promise of this approach, two technical challenges have prevented these generative models from being applied to natural photograph manipulation. First, it is extremely difficult to find a latent code zрқ‘§z to reproduce a given photograph xрқ‘Ҙx with a deep generative model GрқҗәG: xвүҲGвҖӢ(z)xрқҗәz{\textnormal{x}}\approx G({\textnormal{z}}). As shown in В Figure 2b, the reconstructed image GвҖӢ(z)рқҗәzG({\textnormal{z}}) roughly captures the visual content of the input image xрқ‘Ҙx, but visual details are obviously different from the original photo. Second, after manipulation, the newly synthesized pixels from generative models are often incompatible with the existing content from the real image, which makes stitching the new content into the context of the original image challengingВ (Figure 2c).

In this paper, we address the above two issues using an image-specific adaptation method. Our key idea is to learn an image-specific generative model GвҖІвүҲGsuperscriptрқҗәвҖІрқҗәG^{\prime}\approx G, that produces a near-exact solution for our input image x, so that xвүҲGвҖІвҖӢ(z)xsuperscriptрқҗәвҖІz{\textnormal{x}}\approx G^{\prime}({\textnormal{z}}) outside the edited region of the image. We construct our image-specific GвҖІsuperscriptрқҗәвҖІG^{\prime} to share the same semantic representations as of the original GрқҗәG.Importantly, our image-specific GвҖІsuperscriptрқҗәвҖІG^{\prime} produces new visual content, consistent with the original photo while reflecting semantic manipulations (Figure 2d).

(GANs)В (Goodfellow etВ al., 2014) learn to automatically synthesize realistic image samplesВ (MiyatoetВ al., 2018; KarrasetВ al., 2018). GANs have enabled several user-guided image synthesis tasks such as generating images from user sketchesВ (IsolaetВ al., 2017; SangkloyetВ al., 2017), creating face animationВ (Nagano etВ al., 2018; GengetВ al., 2018), synthesizing photos from language descriptionВ (Zhang etВ al., 2017a), inpaintingВ (Yeh etВ al., 2017), and interactively manipulating objects in a generated sceneВ (Bau etВ al., 2019; ParketВ al., 2019). While most of the prior work focuses on generating a new image from scratch given user controls, little work has used GANs for interactively manipulating an existing natural photograph. The key challenge is the mismatch between the GAN-generated content and existing content in the image. Several lines of workВ (Zhu etВ al., 2016; Perarnau etВ al., 2016; BrocketВ al., 2017) propose to manipulate a photo using GANs but only work with a single object (e.g., handbag) at low resolutions (64x64) and often involve post-processing steps. In this work, our method can directly generate a final result and allow semantic manipulations of an entire natural scene.

Manipulating a natural photograph is a central problem in computer graphics and computational photography, where a piece of software manipulates an image to achieve a user-specified goal while keeping the result photorealistic. Example applications include color adjustmentВ (Reinhard etВ al., 2001; LevinetВ al., 2004; An and Pellacini, 2008; XueetВ al., 2012), tone mappingВ (Durand and Dorsey, 2002), image warpingВ (Avidan and Shamir, 2007), image blendingВ (PГ©rezetВ al., 2003; TaoetВ al., 2010), or image reshufflingВ (Barnes etВ al., 2009), to name just a few. These editing tools can modify the low-level features of an image with simple interactions (e.g. scribbles) and work well without external knowledge. On the contrary, to edit the high-level semantics of an image such as the presence and appearance of objects, prior work often demands manual annotations of the object geometryВ (KholgadeetВ al., 2014) and scene layoutВ (KarschetВ al., 2011), choice of an appropriate objectВ (PГ©rezetВ al., 2003; Lalonde etВ al., 2007), RGBD dataВ (ZhangetВ al., 2016a), or hierarchical segmentationsВ (HongetВ al., 2018). Different from those systems, our method allows complex high-level semantic manipulations of natural photos with a simple brush-based user interface. We use natural image statistics learned by a deep generative model to connect simple user interaction to the complex visual world.

Deep learning has achieved compelling results in many image editing tasks. Recent examples include image inpaintingВ (Pathak etВ al., 2016; IizukaetВ al., 2017; YuetВ al., 2018), image colorizationВ (ZhangetВ al., 2016b; IizukaetВ al., 2016), photo stylizationВ (GatysetВ al., 2016; ZhuetВ al., 2017; LietВ al., 2018) and photo enhancementВ (Gharbi etВ al., 2017; Kim and Park, 2018). Learning-based systems allow real-time computation and require no hand-crafted heuristics. Recent work further integrates user interaction into end-to-end learning systems, enabling interactive applications such as user-guided colorizationВ (Zhang etВ al., 2017b) and sketch-based face editingВ (Portenier etВ al., 2018).These methods can achieve high-quality results, but the editing task is fixed at training time and requires specific training data. In this work, we propose an alternative, task-agnostic approach. We learn the natural image statistics using a generative model and allow many different editing applications in the same framework. This enables new visual effects where training data is not available, such as adding objects and changing the appearance of objects.

We propose a general-purpose semantic photo manipulation method that integrates the natural image prior captured by a GAN generator. Figure 2 shows our image editing pipeline. Given a natural photograph as input, we first re-render the image using an image generator. More concretely, to precisely reconstruct the input image, our method not only optimizes the latent representation but also adapts the generator. The user then manipulates the photo using our interactive interface, such as by adding or removing specific objects or changing their appearances. Our method updates the latent representation according to each edit and renders the final result given the modified representation. Our results look both realistic and visually similar to the input natural photograph.

Unfortunately, this method fails to produce visually appealing edits when GрқҗәG is not able to generate images resembling x. Finding a latent code z that can reproduce an arbitrary image xрқ‘Ҙx is hard because, for many images, the range of the generator GрқҗәG does not include any image sufficiently similar to xрқ‘Ҙx in appearance. As shown in Figure 2, existing reconstruction methodsВ (Zhu etВ al., 2016; Dosovitskiy andBrox, 2016) can only roughly re-generate the color and shape of objects in the scene and fail to reproduce the visual details of input images faithfully. Given an inaccurate reconstruction, subsequent edits introduce further artifacts. Besides, the unedited regions of the final result may look different from the input photo due to the reconstruction error.

The image reconstruction problem is hard as it would require the same generator GрқҗәG to be able to reproduce every single detail of every possible input image. And to be useful for incremental editing, new results GвҖӢ(editвҖӢ(z))рқҗәeditzG(\texttt{edit}({\textnormal{z}})) need to be compatible with the input image x as well. By adapting the generator to the original image, our method blends the generated edited image with the unedited parts of the photograph. This scenario is similar to the task of image compositing, so we evaluate the performance of our method in comparison to several compositing approaches. Although our method for adapting a generative network allows various semantic photo manipulations, many challenges remain. First, our method requires an optimization be run after each edit, which takes about 30 seconds on a modern GPU. This time can be reduced by applying fewer optimization steps and incrementally optimizing during editing, or by using a full-image adaptation GwвҖІsubscriptsuperscriptрқҗәвҖІрқ‘ӨG^{\prime}_{w} of the generator that provides a adaptation of the generator that does not depend on the editing region. Although these faster methods introduce more artifacts than our full method, they can provide useful preview of the results at interactive speeds.

Second, another challenge is that latent spaces learned by deep neural networks are not fully disentangled, so finding vector operations to express a user intent can remain challenging. Adding or removing an object from a scene may have some interactions with unrelated objects. Some examples of undesirable interaction are illustrated in Figure 11; in these examples, chairs are reduced by zeroing chair-correlated components in the representation, but the rendered results do not cleanly remove all parts of the chairs, and distorted parts that resemble chair legs remain in the result. We also note that our method for varying the appearances of objects can be brittle; certain classes of objects (such as trees) vary more than other classes (such as domes or skies). For example, in Figure (10) we found that dome shape, but not color, could be varied.

Lastly, the quality and resolution of our current results are still limited by the deep generator that we are usingВ (KarrasetВ al., 2018). For example, in Figure 7, the GAN-synthesized images omit many details such as cars and signage; our method cannot add such objects to an image if they are not modeled by the generator. But our method is not coupled with a specific generative model and will improve with the advancement of models with better quality and resolution (e.g., concurrent workВ (KarrasetВ al., 2019; BrocketВ al., 2019)).

Nevertheless, we have found that in many computer graphics applications the learned natural image prior can help produce more realistic results with less user input. Our work presents a small step towards leveraging the knowledge learned by a deep generative model for semantic photo manipulation tasks.
