A large body of distributed statistical optimization/estimation literature considers ‚Äúone-shot" setting, in which each machine communicates with the server merely once (zhang2013information).In these works, the main objective is to minimize the number of transmitted bits, while keeping the estimation error as low as the error of a centralized estimator, in which the entire data is co-located in the server.
