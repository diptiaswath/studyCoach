Gaussians; however since S𝑆S is dense, computing S​A𝑆𝐴SA is slow. Another matrix which works is a so-called fast Johnson-Lindenstrauss transform, see Sarlós (2006). As in the Gaussian case, S𝑆S has a very small number of rows in the above applications, and computing S​A𝑆𝐴SA can be done in O~​(n​d)~𝑂𝑛𝑑\tilde{O}(nd) time, where O~​(f)~𝑂𝑓\tilde{O}(f) denotes a function of the form f⋅poly​(log⁡f)⋅𝑓poly𝑓f\cdot\mbox{poly}(\log f). For most problems in numerical linear algebra, one needs to at least read all the non-zero entries of A𝐴A, as otherwise one could miss reading a potentially very large entry. For example, in the low rank approximation problem, if there is one entry which is infinite and all other entries are small, the best rank-111 approximation would first fit the single infinite-valued entry. From this perspective, the above nnz​(A)nnz𝐴\mbox{nnz}(A)-time algorithms are optimal. However, there are many applications for which A𝐴A has additional structure. For example, the polynomial interpolation problem is a special case of regression in which the matrix A𝐴A is a Vandermonde matrix. As observed in Avron, Sindhwani, andWoodruff (2013), in this case if S∈ℝpoly​(d/ϵ)×n𝑆superscriptℝpoly𝑑italic-ϵ𝑛S\in\mathbb{R}^{\textrm{poly}(d/\epsilon)\times n} is a CountSketch matrix, then one can compute S​A𝑆𝐴SA in O​(n​log⁡n)+poly​(d/ϵ)𝑂𝑛𝑛poly𝑑italic-ϵO(n\log n)+\mbox{poly}(d/\epsilon) time. The idea of Avron, Sindhwani, andWoodruff (2013) was to simultaneously exploit the sparsity of S𝑆S, together with the fast multiplication algorithm based on the Fast Fourier Transform associated with Vandermonde matrices to reduce the computation of S​A𝑆𝐴SA to a small number of disjoint matrix-vector products. A key fact used in Avron, Sindhwani, andWoodruff (2013) was that submatrices of Vandermonde matrices are also Vandermonde, which is a property that does not hold for other structured families of matrices, such as Toeplitz matrices, which arise in applications like autoregression. There are also sublinear time low rank approximation algorithms of matrices with other kinds of structure, like PSD and distance matrices, see Musco and Woodruff (2017); Bakshi and Woodruff (2018).

An open question, which is the starting point of our work, is if one can extend the results of Avron, Sindhwani, andWoodruff (2013) to any structured matrix A𝐴A. More specifically, can one solve all of the aforementioned linear algebra problems in time T​(A)𝑇𝐴T(A) instead of nnz​(A)nnz𝐴\mbox{nnz}(A), where T​(A)𝑇𝐴T(A) is the time required to compute A​y𝐴𝑦Ay for a single vector y𝑦y?
