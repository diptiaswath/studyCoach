Offline reinforcement learning (Lange etВ al., 2012; Levine etВ al., 2020) refers to the paradigm of learning a policy in the sequential decision making problems, where only the logged data are available and were collected from an unknown environment (Markov Decision Process / MDP). Inspired by the success of scalable supervised learning methods, modern reinforcement learning algorithms (e.g. Silver etВ al. (2017)) incorporate high-capacity function approximators to acquire generalization across large state-action spaces and have achieved excellent performances along a wide range of domains. For instance, there are a huge body of deep RL-based algorithms that tackle challenging problems such as the game of Go and chess (Silver etВ al., 2017; Schrittwieser etВ al., 2020), Robotics (Gu etВ al., 2017; Levine etВ al., 2018), energy control (Degrave etВ al., 2022) and Biology (Mahmud etВ al., 2018; Popova etВ al., 2018). Nevertheless, practitioners also noticed that algorithms with general function approximators can be quite data/sample inefficient, especially for deep neural networks where the models may require million of steps for tuning the large number of parameters they contain.111Check Arulkumaran etВ al. (2017) and the references therein for an overview.

On the other hand, statistical analysis has been actively conducted to understand the sample/statistical efficiency for reinforcement learning with function approximation, and fruitful results have been achieved under the respective model representations (Munos, 2003; Chen and Jiang, 2019; Yang and Wang, 2019; Du etВ al., 2019; Sun etВ al., 2019; Modi etВ al., 2020; Jin etВ al., 2020b; Ayoub etВ al., 2020; Zanette etВ al., 2020; Jin etВ al., 2021a; Du etВ al., 2021; Jin etВ al., 2021b; Zhou etВ al., 2021a; Xie etВ al., 2021a; Min etВ al., 2021; Nguyen-Tang etВ al., 2021; Li etВ al., 2021; Zanette etВ al., 2021; Yin etВ al., 2022; Uehara etВ al., 2022; Cai etВ al., 2022). However, most works consider linear model approximators (e.g. linear (mixture) MDPs) or its variants. While the explicit linear structures make the analysis trackable (linear problems are easier to analyze), they are unable to reveal the sample/statistical complexity behaviors of practical algorithms that apply powerful function approximations (which might have complex structures).

In addition, there is an excellent line of works tackling provably efficient offline RL with general function approximation (e.g. (Chen and Jiang, 2019; Xie etВ al., 2021a; Zhan etВ al., 2022)). Due to the generic function approximation class considered, those complexity bounds are usually expressed in the standard worst-case fashion OвҖӢ(Vmax2вҖӢ1n)рқ‘Ӯsuperscriptsubscriptрқ‘ү21рқ‘ӣO(V_{\max}^{2}\sqrt{\frac{1}{n}}) which lack the characterizations of individual instance behaviors. However, as mentioned in Zanette and Brunskill (2019), practical reinforcement learning algorithms often perform far better than what these problem-independent bounds would suggest.

These observations motivate us to consider function approximation schemes that can help address the existing limitations. In particular, in this work we consider offline reinforcement learning with differentiable function class approximations. Its definition is given in below.

Why consider differentiable function class (DefinitionВ 1.1)? There are two main reasons why differentiable function class is worth studying for reinforcement learning.вҖўDue to the limitation of statistical tools, existing analysis in reinforcement learning usually favor basic settings such as tabular MDPs (where the state space and action space are finite (Azar etВ al., 2013, 2017; Sidford etВ al., 2018; Jin etВ al., 2018; Cui and Yang, 2020; Agarwal etВ al., 2020; Yin etВ al., 2021a, b; Li etВ al., 2020; Ren etВ al., 2021; Xie etВ al., 2021b; Li etВ al., 2022; Zhang etВ al., 2022b; Qiao etВ al., 2022; Cui and Du, 2022)) or linear MDPs (Yang and Wang, 2020; Jin etВ al., 2020b; Wang etВ al., 2020; Jin etВ al., 2021b; Ding etВ al., 2021; Wang etВ al., 2021a; Min etВ al., 2021) / linear Mixture MDPs (Modi etВ al., 2020; Cai etВ al., 2020; Zhang etВ al., 2021a; Zhou etВ al., 2021b, a) (where the transition dynamic admits linear structures) so that well-established techniques (e.g. from linear regression) can be applied. In addition, subsequent extensions are often based on linear models (e.g. Linear Bellman Complete models (Zanette etВ al., 2020) and Eluder dimension (Russo and VanВ Roy, 2013; Jin etВ al., 2021a)). Differentiable function class strictly generalizes over the previous popular choices, i.e. by choosing fвҖӢ(Оё,П•)=вҹЁОё,П•вҹ©рқ‘“рқңғitalic-П•рқңғitalic-П•f(\theta,\phi)=\langle\theta,\phi\rangle or specifying П•italic-П•\phi to be one-hot representations, and is far more expressive as it encompasses nonlinear approximators.вҖўPractically speaking, the flexibility of selecting model fрқ‘“f provides the possibility for handling a variety of tasks. For instance, when fрқ‘“f is specified to be neural networks, Оёрқңғ\theta corresponds to the weights of each network layers and П•вҖӢ(вӢ…,вӢ…)italic-П•вӢ…вӢ…\phi(\cdot,\cdot) corresponds to the state-action representations (which is induced by the network architecture). When facing with easier tasks, we can deploy simpler model fрқ‘“f such as polynomials. Yet, our statistical guarantee is not affected by the specific choices as we can plug the concrete form of model fрқ‘“f into TheoremВ 3.2 to obtain the respective bounds (we do not need separate analysis for different tasks).

Reinforcement learning with function approximation. RL with function approximation has a long history that can date back to Bradtke and Barto (1996); Tsitsiklis and VanВ Roy (1996). Later, it draws significant interest for the finite sample analysis (Jin etВ al., 2020b; Yang and Wang, 2019). Since then, people put tremendous efforts towards generalizing over linear function approximations and examples include Linear Bellman complete models (Zanette etВ al., 2020), Eluder dimension (Russo and VanВ Roy, 2013; Jin etВ al., 2021a), linear deterministic QвӢҶsuperscriptрқ‘„вӢҶQ^{\star} (Wen and VanВ Roy, 2013) or Bilinear class (Du etВ al., 2021). While those extensions are valuable, the structure conditions assumed usually make the classes hard to track beyond the linear case. For example, the practical instances of Eluder Dimension are based on the linear-in-feature (or its transformation) representations (SectionВ 4.1 of Wen and VanВ Roy (2013)). As a comparison, differentiable function class contains a range of functions that are widely used in practical algorithms (Riedmiller, 2005).

Offline RL with general function approximation (GFA). Another interesting thread of work considered offline RL with general function approximation (Ernst etВ al., 2005; Chen and Jiang, 2019; Liu etВ al., 2020; Xie etВ al., 2021a) which only imposes realizability and completeness/concentrability assumptions. The major benefit is that the function hypothesis can be arbitrary with no structural assumptions and it has been shown that offline RL with GFA is provably efficient. However, the generic form of functions in GFA makes it hard to go beyond worst-case analysis and obtain fine-grained instance-dependent learning bounds similar to those under linear cases. On the contrary, our results with DFA can be more problem adaptive by leveraging gradients and higher order information.

In addition to the above, there are more connected works. Zhang etВ al. (2022a) first considers the differentiable function approximation (DFA) for the off-policy evaluation (OPE) task and builds the asymptotic theory, Fan etВ al. (2020) analyzes the deep Q-learning with the specific ReLu activations, and Kallus and Uehara (2020) considers semi-parametric / nonparametric methods for offline RL (as opposed to our parametric DFA in 1.1). These are nice complementary studies to our work.

Motivation of PFQL. The PFQL algorithm mingles the two celebrated algorithmic choices: Fitted Q-Iteration (FQI) and Pessimism. However, before going into the technical details, we provide some interesting insights that motivate our analysis.

First of all, the square error loss used in FQI (Gordon, 1999; Ernst etВ al., 2005) naturally couples with the differentiable function class as the resulting optimization objective is more computationally tractable (since stochastic gradient descent (SGD) can be readily applied) comparing to other information-theoretical algorithms derived with general function approximation (e.g. the maxmin objective in Xie etВ al. (2021a), eqn (3.2)).444We mention Xie etВ al. (2021a) has a nice practical version PSPI, but the convergence is slower (the rate OвҖӢ(nвҲ’13)рқ‘Ӯsuperscriptрқ‘ӣ13O(n^{-\frac{1}{3}})). In particular, FQI with differentiable function approximation resembles the theoretical prototype of neural FQI algorithm (Riedmiller, 2005) and DQN algorithm (Mnih etВ al., 2015; Fan etВ al., 2020) when instantiating the model fрқ‘“f to be deep neural networks. Furthermore, plenty of practical algorithms leverage fitted-Q subroutines for updating the critic step (e.g. (Schulman etВ al., 2017; Haarnoja etВ al., 2018)) with different differentiable function choices.

In addition, we also incorporate pessimism for the design. Indeed, one of the fundamental challenges in offline RL comes from the distributional shift. When such a mismatch occurs, the estimated/optimized Qрқ‘„Q-function (using batch data рқ’ҹрқ’ҹ\mathcal{D}) may witness severe overestimation error due to the extrapolation of model fрқ‘“f (Levine etВ al., 2020). Pessimism is the scheme to mitigate the error / overestimation bias via penalizing the Q-functions at state-action locations that have high uncertainties (as opposed to the optimism used in the online case), and has been widely adopted (e.g. (Buckman etВ al., 2020; Kidambi etВ al., 2020; Jin etВ al., 2021b)).

Feature representation vs. Parameters. One interesting observation from TheoremВ 3.2 is that the learning complexity does not depend on the feature representation dimension mрқ‘ҡm but only on parameter dimension dрқ‘‘d as long as function class в„ұв„ұ\mathcal{F} satisfies differentiability definitionВ 1.1 (not even in the higher order term). This seems to suggest, when changing the model fрқ‘“f with more complex representations, the learning hardness will not grow as long as the number of parameters need to be learned does not increase. Therefore, this heuristic might help people rethink about what is the more essential element (feature representation vs. parameter space) in the representation learning RL regime (e.g. low rank MDPs (Uehara etВ al., 2022)). We leave the concrete understanding the connection between features and parameters as the future work.

In this work, we study offline reinforcement learning with differentiable function approximation and show the sample efficiency for differentiable function learning. We further improve the sample complexity with respect to the horizon dependence via a variance aware variant. However, the dependence of the parameter space still scales with dрқ‘‘d (whereas for linear function approximation this dependence is dрқ‘‘\sqrt{d}), and this is due to applying covering argument for the rich class of differentiable functions. For large deep models, the dimension of the parameter can be huge, therefore it would be interesting to know if certain algorithms can further improve the parameter dependence, or whether this dрқ‘‘d is essential.

Also, how to relax uniform coverage assumptionВ 2.3 is unknown under the current analysis. In addition, due to the technical reason, we require the third-order smoothness in DefinitionВ 1.1. If only the second-order or the first-order derivative information is provided, whether learning efficiency can be achieved remains an interesting question. In addition, understanding the connections between the differentiable function approximation and overparameterized neural networks approximation Nguyen-Tang and Arora (2022); Xu and Liang (2022) is important. We leave these open problems as future work.

Lastly, the differentiable function approximation setting provides a general framework that is not confined to offline RL. Understanding the sample complexity behaviors of online reinforcement learning (Jin etВ al., 2020b; Wang etВ al., 2021b), reward-free learning (Jin etВ al., 2020a; Wang etВ al., 2020) and representation learning (Uehara etВ al., 2022) might provide new and unified views over the existing studies.
