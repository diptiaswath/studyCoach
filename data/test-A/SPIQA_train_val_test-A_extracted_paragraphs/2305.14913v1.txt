The named entity recognition (NER) task aims to locate and classify entity spans in a given text into predefined entity types.It is widely used for many downstream applications, such as relation extraction and question answering.Deep neural networks have made significant progress on this task leveraging large-scale human-annotated data for training.However, fine-grained token-level annotation makes it costly to collect enough high-quality labeled data, especially for low-resource languages.Such scenarios motivate the research on zero-shot cross-lingual NER, which attempts to leverage labeled data in a rich-resource source language to solve the NER task in a target language without annotated data.

Recent attempts at cross-lingual NER can be roughly categorized from two aspects: learning language-independent features via feature alignmentВ (Huang etВ al., 2019; Keung etВ al., 2019) and learning language-specific features from automatically labeled target-language dataВ (Wu etВ al., 2020b, a).Despite bringing great success to cross-lingual NER, the former line of research misses exploiting language-specific features and thus shows substandard performance, especially when transferring to distant languages, e.g., from English to ArabicВ (Fu etВ al., 2023).Hence, a series of studies focuses on the latter category, which typically creates pseudo-labeled target-language data and uses it to perform conventional supervised learning or teacher-student learning.For example, as shown in FigВ 1(a),earlier studies (Ehrmann etВ al., 2011; Mayhew etВ al., 2017; Xie etВ al., 2018; Jain etВ al., 2019), such as TMP Jain etВ al. (2019), first translate labeled data in the source language and then perform label projection.Recently, several approaches have utilized a weak model, which could be an NER model either trained on the source languageвҖҷs labeled data as in TSL Wu etВ al. (2020b), or further finetuned on the generated translation data as in UniTrans Wu etВ al. (2020a), to annotate the unlabeled target-language data for improvement,as shown in FigВ 1(b) and FigВ 1(c).

Unfortunately, these methods inevitably suffer from the label noise induced by inaccurate translation and label projection, or the weak modelвҖҷs limited capability.Although some methods are proposed to mitigate the label noise problem by additionally training an instance selectorВ (Liang etВ al., 2021; Chen etВ al., 2021) or designing heuristic rules for data selectionВ (Ni etВ al., 2017), they independently manipulate either the translation data (рқ’ҹtвҖӢrвҖӢaвҖӢnвҖӢssubscriptрқ’ҹрқ‘Ўрқ‘ҹрқ‘Һрқ‘ӣрқ‘ \mathcal{D}_{trans})В (Ni etВ al., 2017) or the target-language data (рқ’ҹtвҖӢgвҖӢtsubscriptрқ’ҹрқ‘Ўрқ‘”рқ‘Ў\mathcal{D}_{tgt}) pseudo-labeled by NER models trained in the source languageВ (Liang etВ al., 2021; Chen etВ al., 2021).Hence, all these methods ignore the complementary characteristics between both for denoising.Particularly, from the text view, рқ’ҹtвҖӢgвҖӢtsubscriptрқ’ҹрқ‘Ўрқ‘”рқ‘Ў\mathcal{D}_{tgt} is collected from a natural text distribution of the target-language data,while рқ’ҹtвҖӢrвҖӢaвҖӢnвҖӢssubscriptрқ’ҹрқ‘Ўрқ‘ҹрқ‘Һрқ‘ӣрқ‘ \mathcal{D}_{trans} can be regarded as a way of data augmentation to provide more lexicon variants.From the labeling function view, labels of рқ’ҹtвҖӢrвҖӢaвҖӢnвҖӢssubscriptрқ’ҹрқ‘Ўрқ‘ҹрқ‘Һрқ‘ӣрқ‘ \mathcal{D}_{trans} are obtained via the label projection algorithm, which have little association with those of рқ’ҹtвҖӢgвҖӢtsubscriptрқ’ҹрқ‘Ўрқ‘”рқ‘Ў\mathcal{D}_{tgt} generated by NER models.

With such consideration, we propose a model-collaboration-based denoising scheme, which incorporates models trained on both data sources to mutually denoise the pseudo-labels of both data sources in an iterative way.As shown in FigВ 1(d), we first leverage в„іtвҖӢgвҖӢtsubscriptв„ірқ‘Ўрқ‘”рқ‘Ў\mathcal{M}_{tgt} trained on the pseudo-labeled target-language data рқ’ҹtвҖӢgвҖӢtsubscriptрқ’ҹрқ‘Ўрқ‘”рқ‘Ў\mathcal{D}_{tgt} to denoise the translation data annotated by label projection.In this way, the learned model в„іtвҖӢrвҖӢaвҖӢnвҖӢssubscriptв„ірқ‘Ўрқ‘ҹрқ‘Һрқ‘ӣрқ‘ \mathcal{M}_{trans} will be less affected by noise in the translation data.We then employ the improved в„іtвҖӢrвҖӢaвҖӢnвҖӢssubscriptв„ірқ‘Ўрқ‘ҹрқ‘Һрқ‘ӣрқ‘ \mathcal{M}_{trans} to re-label the target-language unlabeled data рқ’ҹtвҖӢgвҖӢtsubscriptрқ’ҹрқ‘Ўрқ‘”рқ‘Ў\mathcal{D}_{tgt}.It is expected that there is less noise in the relabeled data, and thus we can produce a more powerful в„іtвҖӢgвҖӢtsubscriptв„ірқ‘Ўрқ‘”рқ‘Ў\mathcal{M}_{tgt}.We perform this procedure for several iterations, so that all the involved data sources and models can be improved in an upward spiral.

Moreover, borrowing the idea from anomaly detectionВ (Gu etВ al., 2019) that a given data pointвҖҷs neighborhood information can be used to measure its anomalism, here we find that the similar tokens in the feature space can also collaborate for denoising.Previous studiesВ (Zhai and Wu, 2019; Xu etВ al., 2020) have shown that instances with the same label are more likely to locate close to each other in the representation space.Our intuition is that, if a tokenвҖҷs label conflicts a lot with labels of other tokens in its neighborhood, then this label is probably noisy.Therefore, we further propose an instance-collaboration-based denoising strategy to explore the neighborhood structure of each token for denoising, as shown in Figure 2.Specifically, we utilize the label consistency of each tokenвҖҷs neighborhood in the representation space to re-weight the soft-labeled examples in knowledge distillation.

We integrate the instance-collaboration-based denoising strategy into the model-collaboration-based denoising scheme and propose a Collaborative Label Denoising framework, i.e., CoLaDa, for cross-lingual NER.We conduct extensive experiments on two popular benchmarks covering six languages for evaluation.Experimental results show that our method outperforms existing state-of-the-art methods.Qualitative and quantitative analyses further demonstrate the effectiveness of our framework in reducing the data noise.

Previous workВ (Zhai and Wu, 2019; Xu etВ al., 2020) indicates that tokens with the same labels are more likely to locate close to each other in the representation space of a deep neural network.If the label of a given token is inconsistent with lots of its neighbors, this token would be isolated from other tokens with the same label in the feature space, and hence its label is more likely to be noisy.Therefore, we propose instance-collaboration-based denoising, which evaluates the reliability of a given tokenвҖҷs label by measuring the label consistency of its neighborhood, and then uses the reliability score to weight the noisy labels from a teacher model в„ів„і\mathcal{M} for knowledge distillation on data рқ’ҹ={рқ’ҷ}рқ’ҹрқ’ҷ\mathcal{D}=\{\bm{x}\}.Noisy labels are expected to have lower weights than clean ones.

Prior work on cross-lingual NER mainly falls into two major categories: feature-based and data-based transfer.

These methods learn language-independent features so that the model trained on the source language can directly adapt to the target language.Earlier work exploits word clustersВ (TГӨckstrГ¶m etВ al., 2012), gazetteersВ (Zirikly and Hagiwara, 2015), Wikifier featuresВ (Tsai etВ al., 2016), and cross-lingual word embeddingВ (Ni etВ al., 2017), etc.More recently, with the fast growth of multilingual pre-trained language modelsВ (Devlin etВ al., 2019; Conneau etВ al., 2020) and their promising results on cross-lingual transferВ (Wu and Dredze, 2019), lots of studies build upon such pre-trained models and further promote the learning of language-independent features via meta-learningВ (Wu etВ al., 2020c), contrastive alignmentВ (Wu and Dredze, 2020), adversarial learningВ (Keung etВ al., 2019; Chen etВ al., 2021), and by integrating other resourcesВ (Fetahu etВ al., 2022). Despite the great success, they mostly ignore language-specific features, which are especially important when transferring to distant languagesВ (Fu etВ al., 2023).

These approaches learn language-specific features via automatically labeled target-language data and can be further divided into translation-based and knowledge distillation-based methods.

Translation-based methods first translate the source-language data to the target language, then perform label projection from the source side to the target side.Some prior studies have proposed to use cheap translation such as word-to-wordВ (Xie etВ al., 2018) or phrase-to-phraseВ (Mayhew etВ al., 2017) translation. Jain etВ al. (2019) propose an entity projection algorithm to utilize the Google translation system. Recently, Liu etВ al. (2021) and Yang etВ al. (2022) propose to translate sentences with pre-defined markers for label projection. And Ni etВ al. (2017) design heuristic rules to select high-quality translation data.However, both data noise and artifactsВ (Artetxe etВ al., 2020) in the translation data still limit the performance of such methodsВ (GarcГӯa-Ferrero etВ al., 2022).

Knowledge distillation-based methods train a student model on unlabeled target-language data with the soft labels from a teacher modelВ (Wu etВ al., 2020b). Li etВ al. (2022) improve the single task based teacher-student learning with entity similarity as an auxiliary task. To mitigate the label noise from the teacher model, Chen etВ al. (2021) propose AdvPicker, which trains a language discriminator to select the less language-dependent unlabeled data for knowledge distillation; Liang etВ al. (2021) design a reinforcement learning algorithm to train an instance selector according to features such as model confidence to select reliable pseudo labels iteratively.

While most previous work leverage either translation data or unlabeled data, UniTransВ (Wu etВ al., 2020a) utilizes the model trained on translation data to perform teacher-student learning on unlabeled data. But it still suffers from the data noise problem. More recently, consistency trainingВ (Zheng etВ al., 2021; Zhou etВ al., 2022) has also been explored to leverage both unlabeled data and translation data without explicit label annotation.

To the best of our knowledge, we are the first to propose a unified denoising framework to handle data noise in both translation and unlabeled data collaboratively from the model and instance levels for cross-lingual NER.

To address the problem of label noise in cross-lingual NER, this paper presents CoLaDa, a collaborative label denoising framework. We propose a model-collaboration-based denoising scheme to make two models trained on different data sources to denoise the labels of each other and hence promote each otherвҖҷs learning. We further propose an instance-collaboration-based strategy that collaboratively considers the label consistency among similar tokens in the feature space to re-weight the noisy labels assigned by a teacher model in knowledge distillation. By integrating the instance-collaboration strategy into the model-collaboration denoising scheme, our final framework CoLada achieves superior performance over prior start-of-the-art methods by benefiting from better handling the data noise.
