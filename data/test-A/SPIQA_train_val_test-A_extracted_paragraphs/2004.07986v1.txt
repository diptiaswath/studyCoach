Numerical linear algebra algorithms are fundamental building blocks in many machine learning and data mining tasks.A well-studied problem is low rank matrix approximation.The most common version of the problem is also known as Principal Component Analysis (PCA), in which the goal is to find a low rank matrix to approximate a given matrix such that the Frobenius norm of the error is minimized.The optimal solution of this objective can be obtained via the singular value decomposition (SVD).Hence, the problem can be solved in polynomial time.If approximate solutions are allowed, then the running time can be made almost linear in the number of non-zero entries of the given matrix [Sar06, CW13, MM13, NN13, BDN15, Coh16].

An important variant of the PCA problem is the entrywise ℓ1subscriptℓ1\ell_{1}-norm low rank matrix approximation problem.In this problem, instead of minimizing the Frobenius norm of the error, we seek to minimize the ℓ1subscriptℓ1\ell_{1}-norm of the error.In particular, given an n×n𝑛𝑛n\times n input matrix A𝐴A, and a rank parameter k𝑘k, we want to find a matrix B𝐵B with rank at most k𝑘k such that ‖A−B‖1subscriptnorm𝐴𝐵1\|A-B\|_{1} is minimized, where for a matrix C𝐶C, ‖C‖1subscriptnorm𝐶1\|C\|_{1} is defined to be ∑i,j|Ci,j|subscript𝑖𝑗subscript𝐶𝑖𝑗\sum_{i,j}|C_{i,j}|.There are several reasons for using the ℓ1subscriptℓ1\ell_{1}-norm as the error measure.For example, solutions with respect to the ℓ1subscriptℓ1\ell_{1}-norm loss are usually more robust than solutions with Frobenius norm loss [Hub64, CLMW11].Further, the ℓ1subscriptℓ1\ell_{1}-norm loss is often used as a relaxation of the ℓ0subscriptℓ0\ell_{0}-loss, which has wide applications including sparse recovery, matrix completion, and robust PCA; see e.g., [XCS10, CLMW11].Although a number of algorithms have been proposed for the ℓ1subscriptℓ1\ell_{1}-norm loss [KK03, KK05, KLC+15, Kwa08, ZLS+12, BJ12, BD13, BDB13, MXZZ13, MKP13, MKP14, MKCP16, PK16], the problem is known to be NP-hard [GV15].The first ℓ1subscriptℓ1\ell_{1}-low rank approximation with provable guarantees was proposed by [SWZ17].To cope with NP-hardness, the authors gave a solution with a poly⁡(k​log⁡n)poly𝑘𝑛\operatorname{poly}(k\log n)-approximation ratio, i.e., their algorithm outputs a rank-k𝑘k matrix B′∈ℝn×nsuperscript𝐵′superscriptℝ𝑛𝑛B^{\prime}\in\mathbb{R}^{n\times n} for which‖A−B′‖1≤α⋅minrank−k​B⁡‖A−B‖1subscriptnorm𝐴superscript𝐵′1⋅𝛼subscriptrank𝑘𝐵subscriptnorm𝐴𝐵1\displaystyle\|A-B^{\prime}\|_{1}\leq\alpha\cdot\min_{\operatorname{rank}-k\ B}\|A-B\|_{1}(1)for α=poly⁡(k​log⁡n)𝛼poly𝑘𝑛\alpha=\operatorname{poly}(k\log n).The approximation ratio α𝛼\alpha was further improved to O​(k​log⁡k)𝑂𝑘𝑘O(k\log k) by allowing B′superscript𝐵′B^{\prime} to have a slightly larger k′=O​(k​log⁡n)superscript𝑘′𝑂𝑘𝑛k^{\prime}=O(k\log n) rank [CGK+17]. Such B′superscript𝐵′B^{\prime} with larger rank is referred to as a bicriteria solution.However, in high precision applications, such approximation factors are too large.A natural question is if one can compute a (1+ϵ)1italic-ϵ(1+\epsilon)-approximate solution efficiently for ℓ1subscriptℓ1\ell_{1}-norm low rank approximation.In fact, a (1+ϵ)1italic-ϵ(1+\epsilon)-approximation algorithm was given in [BBB+19], but the running time of their algorithm is a prohibitive npoly⁡(k/ϵ)superscript𝑛poly𝑘italic-ϵn^{\operatorname{poly}(k/\epsilon)}.Unfortunately, [BBB+19] shows in the worst case that a 2kΩ​(1)superscript2superscript𝑘Ω12^{k^{\Omega(1)}} running time is necessary for any constant approximation given a standard conjecture in complexity theory.
