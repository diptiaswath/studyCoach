Spatio-temporal point process (STPP) is a stochastic collection of points, where each point denotes an event x=(t,s)рқ‘Ҙрқ‘Ўрқ‘ x=(t,s) associated with time tрқ‘Ўt and location sрқ‘ s.STPP is a principled framework for modeling sequences consisting of spatio-temporal events, and has been applied in a wide range of fields, such as earthquakes and aftershocksВ (Ogata, 1988; Chen etВ al., 2021), disease spreadВ (Meyer etВ al., 2012; Park etВ al., 2022), urban mobilityВ (Wang etВ al., 2021; Yuan etВ al., 2023, 2022; Long etВ al., 2023), and emergenciesВ (Xu etВ al., 2016; Zhu etВ al., 2021).

Spatio-temporal point processes have been widely studied in the literatureВ (Diggle, 2006; GonzГЎlez etВ al., 2016; Reinhart, 2018; Baddeley etВ al., 2007; Chen etВ al., 2021; Zhuang, 2006; Yang etВ al., 2018) with rich theoretical foundationsВ (Daley etВ al., 2003; Grandell, 2012; Kingman, 1992).Due to computational complexities, a general approach for STPPs is to characterize the event time and space with distinct models. Conventional STPP modelsВ (Diggle, 2006; GonzГЎlez etВ al., 2016; Reinhart, 2018) mainly capture relatively simple patterns of spatio-temporal dynamics, where the temporal domain is modeled by temporal point process models, such as Poisson processВ (Kingman, 1992), Hawkes processВ (Hawkes, 1971), and Self-correcting processВ (Isham and Westcott, 1979), and the spatial domain is usually fitted by kernel density estimators (KDE)В (W\keglarczyk, 2018).With the advance of neural networks, a series of neural architectures are proposed to improve the fitting accuracyВ (Jia and Benson, 2019; Chen etВ al., 2021; Zhou etВ al., 2022). However, they still adopt the approach of separate modeling.For example, Chen et al.В (Chen etВ al., 2021) use neural ODEs and continuous-time normalizing flows (CNFs) to learn the temporal distribution and spatial distribution, respectively. Zhou et al.В (Zhou etВ al., 2022) apply two independent kernel functions for time and space, whose parameters are obtained from neural networks, to build the density function.

However, for STPPs, the time and space where an event occurs are highly dependent and entangled with each other.For example, in seismology, earthquakes are spatio-temporal correlated due to crust movementsВ (Wang etВ al., 2017), which occur with a higher probability close in time and space to previous earthquakes.Take urban mobility as another example, people are more likely to go to work during the day, while tend to go for entertainment at night.Therefore, it is crucial to learn models that can address the spatio-temporal joint distribution conditioned on the event history.However, it is non-trivial due to the following two challenges:

Spatio-temporal Point Processes.Temporal point process modelsВ (Du etВ al., 2016; Mei and Eisner, 2017; Zuo etВ al., 2020; Zhang etВ al., 2020; Lin etВ al., 2022) can be directly used for STPPs, where the space is considered as the event marker. Kernel density estimation methods are also used to model continuous-space distributions in STPP modelsВ (Chen etВ al., 2021; Zhou etВ al., 2022; Reinhart, 2018; Jia and Benson, 2019; Moller and Waagepetersen, 2003; Baddeley etВ al., 2007). Most existing solutions follow an intensity-based paradigm, and their main challenge is how to choose a good parametric form for the intensity function. There exists a trade-off between the modeling capability of the intensity function and the cost to compute the log-likelihood. Some intensity-free modelsВ (Omi etВ al., 2019; Shchur etВ al., 2019) are proposed to tackle this problem; however, the probability density function either is unavailableВ (Omi etВ al., 2019) or still has certain model restrictionsВ (Shchur etВ al., 2019). Another drawback of existing models is that they can only model either the continuous-space domain or the discrete-space domain, which largely limits their usability in real-world scenarios.

Recently, a line of advances have been developed for the generative modeling of point processes. For example, generative adversarial networksВ (Xiao etВ al., 2017; Dizaji etВ al., 2022) are used to learn to generate point processes in a likelihood-free manner. Reinforcement learningВ (Li etВ al., 2018; Upadhyay etВ al., 2018) approaches and variational autoencodersВ (Mehrasa etВ al., 2019; Pan etВ al., 2020) are also included to explore the generative performance of TPPs. Some works also use noise contrastive learningВ (Guo etВ al., 2018; Mei etВ al., 2020) instead of MLE. We are the first to learn point processes within the paradigm of diffusion models, which successfully address limitations in previous existing solutions.

Denoising Diffusion Probabilistic Models.Denoising diffusion probabilistic models (DDPM)В (Sohl-Dickstein etВ al., 2015; Ho etВ al., 2020; Song etВ al., 2020), are a class of deep generative models, which are inspired by non-equilibrium thermodynamics. Due to their powerful generative capabilities, diffusion models have been used in a wide range of applications including image generationВ (Dhariwal and Nichol, 2021; Austin etВ al., 2021; Sinha etВ al., 2021; Rombach etВ al., 2022), time series prediction and imputationВ (Rasul etВ al., 2021; Tashiro etВ al., 2021), audio generationВ (Goel etВ al., 2022; Ho etВ al., 2022; Kong etВ al., 2020), text generationВ (Li etВ al., 2022a; Gong etВ al., 2022; Gao etВ al., 2022), 3D point cloud generationВ (Luo and Hu, 2021; Zhou etВ al., 2021), and trajectory generationВ (Gu etВ al., 2022; Pekkanen etВ al., 2021). In this paper, we first introduce the diffusion model to the domain of spatio-temporal point processes.

The framework also poses desired properties like easy training and closed-form sampling.
