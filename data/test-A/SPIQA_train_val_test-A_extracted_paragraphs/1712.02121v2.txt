Large-scale knowledge bases (KBs), such as YAGO (Suchanek etВ al., 2007), Freebase (Bollacker etВ al., 2008) and DBpedia (Lehmann etВ al., 2015), are usually databases of triples representing the relationships between entities in the form of fact (head entity, relation, tail entity) denoted as (h, r, t), e.g., (Melbourne, cityOf, Australia).These KBs are useful resources in many applications such as semantic searching and ranking (Kasneci etВ al., 2008; Schuhmacher and Ponzetto, 2014; Xiong etВ al., 2017), question answering (Zhang etВ al., 2016; Hao etВ al., 2017) and machine reading (Yang and Mitchell, 2017).However, the KBs are still incomplete, i.e., missing a lot of valid triples (Socher etВ al., 2013; West etВ al., 2014).Therefore, much research work has been devoted towards knowledge base completion or link prediction to predict whether a triple (h, r, t) is valid or not (Bordes etВ al., 2011).

Many embedding models have proposed to learn vector or matrix representations for entities and relations, obtaining state-of-the-art (SOTA) link prediction results Nickel etВ al. (2016a). In these embedding models, valid triples obtain lower implausibility scores than invalid triples.Let us take the well-known embedding model TransE (Bordes etВ al., 2013) as an example. In TransE, entities and relations are represented by kрқ‘ҳk-dimensional vector embeddings.TransE employs a transitional characteristic to model relationships between entities, in which it assumes that if (h, r, t) is a valid fact, the embedding of head entity hв„Һh plus the embedding of relation rрқ‘ҹr should be close to the embedding of tail entity tрқ‘Ўt, i.e. рқ’—hsubscriptрқ’—в„Һ\boldsymbol{v}_{h} + рқ’—rsubscriptрқ’—рқ‘ҹ\boldsymbol{v}_{r} вүҲ\approx рқ’—tsubscriptрқ’—рқ‘Ў\boldsymbol{v}_{t}(here, рқ’—hsubscriptрқ’—в„Һ\boldsymbol{v}_{h}, рқ’—rsubscriptрқ’—рқ‘ҹ\boldsymbol{v}_{r} and рқ’—tsubscriptрқ’—рқ‘Ў\boldsymbol{v}_{t} are embeddings of hв„Һh, rрқ‘ҹr and tрқ‘Ўt respectively). That is, a TransE score вҖ–рқ’—h+рқ’—rвҲ’рқ’—tвҖ–ppsuperscriptsubscriptnormsubscriptрқ’—в„Һsubscriptрқ’—рқ‘ҹsubscriptрқ’—рқ‘Ўрқ‘қрқ‘қ\|\boldsymbol{v}_{h}+\boldsymbol{v}_{r}-\boldsymbol{v}_{t}\|_{p}^{p} of the valid triple (h, r, t) should be close to 00 and smaller than a score вҖ–рқ’—hвҖІ+рқ’—rвҖІвҲ’рқ’—tвҖІвҖ–ppsuperscriptsubscriptnormsubscriptрқ’—superscriptв„ҺвҖІsubscriptрқ’—superscriptрқ‘ҹвҖІsubscriptрқ’—superscriptрқ‘ЎвҖІрқ‘қрқ‘қ\|\boldsymbol{v}_{h^{\prime}}+\boldsymbol{v}_{r^{\prime}}-\boldsymbol{v}_{t^{\prime}}\|_{p}^{p} of an invalid triple (hвҖҷ, rвҖҷ, tвҖҷ).The transitional characteristic in TransE also implies the global relationships among same dimensional entries of рқ’—hsubscriptрқ’—в„Һ\boldsymbol{v}_{h}, рқ’—rsubscriptрқ’—рқ‘ҹ\boldsymbol{v}_{r} and рқ’—tsubscriptрқ’—рқ‘Ў\boldsymbol{v}_{t}.

Other transition-based models extend TransEto additionally use projection vectors or matrices to translate head and tail embeddings into the relation vector space, such as: TransH (Wang etВ al., 2014), TransR (Lin etВ al., 2015b), TransD (Ji etВ al., 2015), STransE (Nguyen etВ al., 2016b) and TranSparse (Ji etВ al., 2016).Furthermore, DISTMULT (Yang etВ al., 2015) and ComplEx (Trouillon etВ al., 2016) use a tri-linear dot product to compute the score for each triple.Recent research has shown that using relation paths between entities in the KBs could help to get contextual information for improving KB completion performance (Lin etВ al., 2015a; Luo etВ al., 2015; Guu etВ al., 2015; Toutanova etВ al., 2016; Nguyen etВ al., 2016a). See other embedding models for KB completion in Nguyen (2017).

Recently, convolutional neural networks (CNNs), originally designed for computer vision (LeCun etВ al., 1998), have significantly received research attention in natural language processing (Collobert etВ al., 2011; Kim, 2014).CNN learns non-linear features to capture complex relationships with a remarkably less number of parameters compared to fully connected neural networks.Inspired from the success in computer vision, Dettmers etВ al. (2018) proposed ConvEвҖ”the first model applying CNN for the KB completion task.In ConvE, only рқ’—hsubscriptрқ’—в„Һ\boldsymbol{v}_{h} and рқ’—rsubscriptрқ’—рқ‘ҹ\boldsymbol{v}_{r} are reshaped and then concatenated into an input matrix which is fed to the convolution layer.Different filters of the same 3Г—3333\times 3 shape are operated over the input matrix to output feature map tensors. These feature map tensors are then vectorized and mapped into a vector via a linear transformation.Then this vector is computed with рқ’—tsubscriptрқ’—рқ‘Ў\boldsymbol{v}_{t} via a dot product to return a score for (h, r, t). See a formal definition of the ConvE score function in Table 1.It is worth noting that ConvE focuses on the local relationships among different dimensional entries in each of рқ’—hsubscriptрқ’—в„Һ\boldsymbol{v}_{h} or рқ’—rsubscriptрқ’—рқ‘ҹ\boldsymbol{v}_{r}, i.e., ConvE does not observe the global relationships among same dimensional entries of an embedding triple (рқҗҜhsubscriptрқҗҜв„Һ\boldsymbol{v}_{h}, рқҗҜrsubscriptрқҗҜрқ‘ҹ\boldsymbol{v}_{r}, рқҗҜtsubscriptрқҗҜрқ‘Ў\boldsymbol{v}_{t}), so that ConvE ignores the transitional characteristic in transition-based models, which is one of the most useful intuitions for the task.

In this paper, we present ConvKBвҖ”an embedding model which proposes a novel use of CNN for the KB completion task. In ConvKB, each entity or relation is associated with an unique kрқ‘ҳk-dimensional embedding. Let рқ’—hsubscriptрқ’—в„Һ\boldsymbol{v}_{h}, рқ’—rsubscriptрқ’—рқ‘ҹ\boldsymbol{v}_{r} and рқ’—tsubscriptрқ’—рқ‘Ў\boldsymbol{v}_{t} denote kрқ‘ҳk-dimensional embeddings of hв„Һh, rрқ‘ҹr and tрқ‘Ўt, respectively.For each triple (h, r, t), the corresponding triple of kрқ‘ҳk-dimensional embeddings (рқ’—hsubscriptрқ’—в„Һ\boldsymbol{v}_{h}, рқ’—rsubscriptрқ’—рқ‘ҹ\boldsymbol{v}_{r}, рқ’—tsubscriptрқ’—рқ‘Ў\boldsymbol{v}_{t}) is represented as a kГ—3рқ‘ҳ3k\times 3 input matrix.This input matrix is fed to the convolution layer where different filters of the same 1Г—3131\times 3 shape are used to extract the global relationships among same dimensional entries of the embedding triple.That is, these filters are repeatedly operated over every row of the input matrix to produce different feature maps. The feature maps are concatenated into a single feature vector which is then computed with a weight vector via a dot product to produce a score for the triple (h, r, t).This score is used to infer whether the triple (h, r, t) is valid or not.

Our contributions in this paper are as follows:вҖўWe introduce ConvKBвҖ”a novel embedding model of entities and relationships for knowledge base completion.ConvKB models the relationships among same dimensional entries of the embeddings. This implies that ConvKB generalizes transitional characteristics in transition-based embedding models.вҖўWe evaluate ConvKB on two benchmark datasets: WN18RR (Dettmers etВ al., 2018) and FB15k-237 (Toutanova and Chen, 2015). Experimental results show that ConvKB obtains better link prediction performance than previous SOTA embedding models. In particular, ConvKB obtains the best mean rank and the highest Hits@10 on WN18RR, and produces the highest mean reciprocal rank and highest Hits@10 on FB15k-237.

In the KB completion or link prediction task (Bordes etВ al., 2013), the purpose is to predict a missing entity given a relation and another entity, i.e, inferring hв„Һh given (r,t)рқ‘ҹрқ‘Ў(r,t) or inferring tрқ‘Ўt given (h,r)в„Һрқ‘ҹ(h,r).The results are calculated based on ranking the scores produced by the score function fрқ‘“f on test triples.

Following Bordes etВ al. (2013), for each valid test triple (h,r,t)в„Һрқ‘ҹрқ‘Ў(h,r,t), we replace either hв„Һh or tрқ‘Ўt by each of other entities in в„°в„°\mathcal{E} to create a set of corrupted triples.We use the вҖңFilteredвҖқ setting protocol (Bordes etВ al., 2013), i.e., not taking any corrupted triples that appear in the KB into accounts.We rank the valid test triple and corrupted triples in ascending order of their scores.We employ three common evaluation metrics: mean rank (MR), mean reciprocal rank (MRR), and Hits@10 (i.e., the proportion of the valid test triples ranking in top 10 predictions).Lower MR, higher MRR or higher Hits@10 indicate better performance.

In this paper, we propose a novel embedding model ConvKB for the knowledge base completion task.ConvKB applies the convolutional neural network to explore the global relationships among same dimensional entries of the entity and relation embeddings, so that ConvKB generalizes the transitional characteristics in the transition-based embedding models.Experimental results show that our model ConvKB outperforms other state-of-the-art models on two benchmark datasets WN18RR and FB15k-237.
