Provably Lipschitz convolutional neural networks:The class of fully connected neural networks (FCNs) which are Gradient Norm Preserving (GNP) and 1-Lipschitz were first introduced by Anil etВ al. (2018). They orthogonalize weight matrices and use GroupSortGroupSort\mathrm{GroupSort} as the activation function to design each layer to be GNP. While there have been numerous works on enforcing Lipschitz constraints on convolution layers (CissГ© etВ al., 2017; Tsuzuku etВ al., 2018; Qian & Wegman, 2019; Gouk etВ al., 2020; Sedghi etВ al., 2019), they either enforce loose Lipschitz bounds or are not scalable to large networks. To ensure that the Lipschitz constraint on convolutional layers is tight, multiple recent works try to construct convolution layers with an orthogonal Jacobian matrix (Li etВ al., 2019b; Trockman & Kolter, 2021; Singla & Feizi, 2021). These approaches avoid the aforementioned issues and allow the training of large, provably 111-Lipschitz CNNs while achieving impressive results.

Provable defenses against adversarial examples: A provably robust classifier is one for which we can guarantee that the classifierвҖҷs prediction remains constant within some region around the input. Most of the existing methods for provable robustness either bound the Lipschitz constant of the neural network or the individual layers (Weng etВ al., 2018; Zhang etВ al., 2019; 2018; Wong etВ al., 2018; Wong & Kolter, 2018; Raghunathan etВ al., 2018; Croce etВ al., 2019; Singh etВ al., 2018; Singla & Feizi, 2020; Zhang etВ al., 2021; 2022; Wang etВ al., 2021; Huang etВ al., 2021). However, these methods do not scale to large and practical networks on the ImageNet dataset (Deng etВ al., 2009). To scale to such large networks, randomized smoothing (Liu etВ al., 2018; Cao & Gong, 2017; LГ©cuyer etВ al., 2018; Li etВ al., 2019a; Cohen etВ al., 2019; Salman etВ al., 2019; Levine etВ al., 2019; Kumar etВ al., 2020a; b) has been proposed as a probabilistically certified defense. However, certifying robustness with high probability requires generating a large number of noisy samples leading to high inference-time computation. In contrast, the defense we propose is deterministic and hence not comparable to randomized smoothing. While Levine & Feizi (2021) provide deterministic robustness certificates using randomized smoothing, their certificates are in the l1subscriptрқ‘ҷ1l_{1} norm and not directly applicable for the l2subscriptрқ‘ҷ2l_{2} threat model studied in this work. Since each group has 222 linear regions, we get 2m/2superscript2рқ‘ҡ22^{m/2} linear regions from the m/2рқ‘ҡ2m/2 groups. Thus, to increase the expressive power, we similarly divide рқҗірқҗі\mathbf{z} into m/2рқ‘ҡ2m/2 groups of size 222 each and apply the 222-dimensional Householder activation function of Order 111 (ПғОё)subscriptрқңҺрқңғ(\sigma_{\theta}) to each group resulting in 2m/2superscript2рқ‘ҡ22^{m/2} linear regions (same as MaxMinMaxMin\mathrm{MaxMin}).

Each of these tensors is of size m/2Г—nГ—nрқ‘ҡ2рқ‘ӣрқ‘ӣm/2\times n\times n giving n2вҖӢm/2superscriptрқ‘ӣ2рқ‘ҡ2n^{2}m/2 groups. We use the same Оёрқңғ\theta for each pair of channels (irrespective of spatial location) resulting in m/2рқ‘ҡ2m/2 learnable parameters. We initialize each Оё=ПҖ/2рқңғрқңӢ2\theta=\pi/2 so that ПғОёsubscriptрқңҺрқңғ\sigma_{\theta} is equivalent to MaxMinMaxMin\mathrm{MaxMin} at initialization.

In this work, we introduce a procedure to certify robustness of 111-Lipschitz convolutional neural networks without orthogonalizing of the last linear layer of the network. Additionally, we introduce a certificate regularization that significantly improves the provable robust accuracy for these models at higher l2subscriptрқ‘ҷ2l_{2} radii. Finally, we introduce a class of GNP activation functions called Householder (or HHHH\mathrm{HH}) activations and prove that the Jacobian of any Gradient Norm Preserving (GNP) piecewise linear function is only allowed to change via Householder transformations for the function to be continuous which provides a general principle for designing piecewise linear GNP functions. These ideas lead to improved deterministic в„“2subscriptв„“2\ell_{2} robustness certificates on CIFAR-10 and CIFAR-100.
