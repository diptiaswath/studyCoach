In automatic speech recognitionВ (ASR) with the hybrid neural network/hidden Markov modelВ (NN/HMM) approachВ [1], the co-articulation effect is addressed by considering tied allophone states in right and left context, known also as senones.В The parameter tying relies generally on the state clustering done by classification and regression treesВ (CART)В [2], and requires a frame-wise alignment of the input speech signal to a hidden HMM state sequence.В This alignment, can be obtained from a Gaussian Mixture ModelВ (GMM) system, and is used for both estimation of CART and frame-wise cross-entropyВ (CE) training of the neural network.В The NN component is used for the substitution of the likelihood of a given feature vector by the normalized scaled posterior probability over the tied state label.В The overall system has separate components and requires different sources of knowledge.В Moreover, having tied state labels as the NN target can lead to overfitting problems due to the arbitrary choices done for the estimation of the clusteringВ [3].

In order to simplify the described pipeline, the recent research focus has shifted to the joint optimization of the different components in a sequence to sequenceВ (seq2seq) framework.В In this approach, the speech sequence is directly mapped to the final word sequence, with an optional use of an external LM.В In connectionist temporal classificationВ (CTC)В [4], the necessity of having an initial frame-wise alignment is relaxed by introducing a blank label and using a sequence-level CE which sums over all possible alignments.В The CTC criterion contains an independence assumption at each time frame.В The inclusion of the dependency to the full or truncated context is introduced with Recurrent Neural Network TransducersВ (RNN-T)В [5] and Recurrent Neural AlignerВ [6] with different constraints in how to traverse the alignment lattice.В Other prior work focuses on the elimination of state-tying for both hybrid HMM and CTC frameworks using full set of diphone statesВ [7] or a triphone representation via an embedding networkВ [8].В The aforementioned seq2seq models, are similar to the hybrid approach in providing a statistical model of the posterior distribution over different label topologies and units.В This posterior is later used in a (weighted) finite state transducer like structure for decoding.В However, they do not use a local normalization and the correct way to combine an external LM is still an open questionВ [9, 10, 11].В The modular formulation in the hybrid approach results in an intrinsic robustness for tasks such as domain adaptation and for low resource ASR tasksВ [12].В Furthermore, seq2seq models require large amounts of data and many additional regularization techniques in order to reach a reasonable performance.В Nevertheless, hybrid models with alternative label units and encoder architecturesВ [13, 14] can still have competitive performance compared to highly optimized seq2seq pipelinesВ [15, 16].В We believe that the first step for a possible simplification of the standard hybrid pipeline is the elimination of the state-tying algorithm.В Recently, we presented a factored hybrid HMM modelВ [17].В Given the complete set of triphone states, the factored model learns to predict context-dependent posteriors over phonemes, in a multi-task manner.В This is achieved by a decomposition of the joint label identity of the center phoneme state and its left-right phonemes.В With this previous work, and its extensionВ [18], we have shown that it is possible to eliminate the state-tying for the determination of the state inventory, and obtain similar performance to a hybrid CART.

Our main contribution with this work is three-fold: (1) We eliminate the dependency to the clustering algorithm entirely by proposing a from scratch pipeline.В With that we obtain slightly better performance compared to the standard hybrid using state-tying, (2) We show that our factored hybrid outperforms the standard approach when using the same initial alignment. And, (3) we show that our modeling approach can obtain larger improvement from regularization techniques, also on a small task such as 300 hours SwitchboardВ (SWB), compared to the CART based hybrid.В We also present a set of baseline results using LibriSpeechВ (LBS), without the main regularization techniques, confirming the scalability of our approach to a larger task.

The common starting point for all experiments is the initialization of the parameters of a context-independent single Gaussian HMM based on a linear segmentation of the input signal.В Using the resulting alignment, a GMM monophone system is bootstrapped.В We denote the resulting alignment as GMM-Mono.В For the CART system, the set of context-dependent tied states are first clustered in order to build a GMM triphone system.В After the inclusion of the speaker adaptation, another CART for clustering the final HMM state inventory is estimated.В Each described step consists of different iterations of parameter re-estimation and subsequent re-alignments.В We denote the intermediate alignment obtained from the speaker-adapted triphone system as GMM-Tri.В For the final best alignment, which we denote as Tandem, first a tandem model with speaker adaptation is trained, followed by miniumum phone error sequence discriminative trainingВ [19].В All the described steps above are necessary for the CART based system in order to reach a good performance.В For each of the alignments described above, we compare Word Error RatesВ (WERs) of our factored hybrid models.В We use GMM-Mono and Tandem alignments for Switchboard 300h, and GMM-{Mono,Triphone} alignments for Librispeech 960h.

The triphone factored hybrid, as described in Eq.В 1a has three separate softmax layers: two of them have as target the left and right phonemes and the third is over the center phoneme states.В The context-dependency for the right phoneme and center state is fed via a soft embedding.В The overall training procedure consists of multiple stages.В In each stage, we increase the model complexity as follows.В We start with a monophone model without feeding any context.В In the following stages, we increase the context-dependency by feeding the context via phoneme and phoneme state embeddingsВ [17].В The effect of the multi-stage training is reported in Table 1.

In addition to the common regularization techniques described later in 3.1, we focused on the effect of the following two techniques.В Label SmoothingВ (LS) is usually used in order to contrast the modelвЂ™s overconfidence about the aligned targetВ [20].В The minimization of the cross-entropy in LS is done with respect to a soft target.В The target is obtained by interpolating a uniform distribution over other classes with the aligned target.В Differently to the monophone targets, where the problem is well-defined, the application of LS is less straightforward when context-dependent targets are considered.В Since the posterior over the untied triphone state corresponds to the combination of the factored posteriors, the idea is to keep a hard target for the center phoneme state and apply LS only on the context.В Consequently, starting from the diphone stage we turn off LS on the center state output.В Experimental results, as reported in Table 2, for a CART-free pipeline starting with the GMM-Mono alignment confirm the following: (1) The relative improvement obtained from LS in monophone model is larger than in di- and triphone models, (2) continuing with LS on all outputs leads to degradation for diphone model.В For CART based models application of LS did not help.В In this case, each label corresponds to a set of tied triphone states, and therefore the model learns to discriminate between state classes defined by the clustering and not by the triphone state identity itself.В We also tried Specaugment, based on random on-the-fly time and feature masking used widely in seq2seq trainingВ [21].В All models are trained with the same masking parameters, following indications fromВ [22].

In this paper, we have shown that the elimination of explicit state-tying from hybrid HMM models results in performance gains across multiple tasks.В Our factored hybrid approach works better in combination with popular regularization techniques compared to a hybrid system using state-tying.В This work also demonstrates that the hybrid approach is still competitive with nowadays popular sequence to sequence models.
