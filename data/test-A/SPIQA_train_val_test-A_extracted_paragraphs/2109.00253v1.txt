Pre-trained language models like BERT В (Devlin etВ al., 2019)and GPT В (Radford and Narasimhan, 2018)have achieved phenomenal successes on a wide range of NLP tasks.However,sentence representations for different languages are not very well aligned,even for pre-trained multilingual modelssuch as mBERT В (Pires etВ al., 2019; Wang etВ al., 2020).This issue is more prominent for language pairs from different families(e.g., English versus Chinese).Also,previous work В (Li etВ al., 2020) has shown that the out-of-box BERT embeddingsperform poorly on monolingual semantic textual similarity (STS) tasks.

There are two general goals for sentence representation learning:cross-lingual representations should be aligned,which is a crucial step for taskslike bitext mining В (Artetxe and Schwenk, 2019a),unsupervised machine translation В (Lample etВ al., 2018b),and zero-shot cross-lingual transfer В (Hu etВ al., 2020) etc.Another goal is to induce a metric space,where semantic similarities can be computed with simple functions(e.g., dot product on L2subscriptрқҗҝ2L_{2}-normalized representations).

Translation ranking В (Feng etВ al., 2020; Yang etВ al., 2020)can serve as a surrogate task to align sentence representations.Intuitively speaking,parallel sentences should have similar representations and are therefore ranked higher,while non-parallel sentences should have dissimilar representations.Models are typically trained with in-batch negatives,which need a large batch size to alleviate the easy negatives issue В (Chen etВ al., 2020a).В Feng etВ al. (2020) use cross-accelerator negative samplingto enlarge the batch size to 204820482048 with 323232 TPU cores.Such a solution is hardware-intensive and still struggles to scale.

Momentum Contrast (MoCo) В (He etВ al., 2020) decouples the batch sizeand the number of negatives by maintaining a large memory queue and a momentum encoder.MoCo requires that queries and keys lie in a shared input space.In self-supervised vision representation learning,both queries and keys are transformed image patches.However,for translation ranking task,the queries and keys come from different input spaces.In this paper,we present dual momentum contrast to solve this issue.Dual momentum contrast maintains two memory queuesand two momentum encoders for each language.It combines two contrastive losses by performing bidirectional matching.

We conduct experiments on the English-Chinese language pair.Language models that are separately pre-trained for English and Chineseare fine-tuned using translation ranking task with dual momentum contrast.To demonstrate the improved quality of the aligned sentence representations,we report state-of-the-art resultson both cross-lingual and monolingual evaluation datasets:Tatoeba similarity search dataset (accuracy 95.9%вҶ’97.4%вҶ’percent95.9percent97.495.9\%\rightarrow 97.4\%),BUCC 2018 bitext mining dataset (f1 score 92.27%вҶ’93.66%вҶ’percent92.27percent93.6692.27\%\rightarrow 93.66\%),and 7 English STS datasets (average SpearmanвҖҷs correlation 77.07%вҶ’78.95%вҶ’percent77.07percent78.9577.07\%\rightarrow 78.95\%).We also carry out several ablation studies to help understandthe learning dynamics of our proposed model.

Multilingual representation learning aims to jointly model multiple languages.Such representations are crucial for multilingual neural machine translation В (Aharoni etВ al., 2019),zero-shot cross-lingual transfer В (Artetxe and Schwenk, 2019b),and cross-lingual semantic retrieval В (Yang etВ al., 2020) etc.Multilingual BERT В (Pires etВ al., 2019) simply pre-trainson the concatenation of monolingual corporaand shows good generalization for taskslike cross-lingual text classification В (Hu etВ al., 2020).Another line of work explicitly aligns representations from language-specific models,either unsupervised В (Lample etВ al., 2018a) or supervised В (Reimers and Gurevych, 2020; Feng etВ al., 2020).

Contrastive learning works by pulling positive instances closer and pushing negatives far apart.It has achieved great successes in self-supervised vision representation learning,including SimCLR В (Chen etВ al., 2020a),MoCo В (He etВ al., 2020; Chen etВ al., 2020b),BYOL В (Grill etВ al., 2020),CLIP В (Radford etВ al., 2021) etc.Recent efforts introduced contrastive learning into various NLP tasksВ (Xiong etВ al., 2020; Giorgi etВ al., 2020; Chi etВ al., 2021; Gunel etВ al., 2020).Concurrent to our work,SimCSE В (Gao etВ al., 2021) uses dropout and hard negatives from NLI datasetsfor contrastive sentence similarity learning,Sentence-T5 В (Ni etВ al., 2021) outperforms SimCSE by scaling to larger models,and xMoCo В (Yang etВ al., 2021) adopts a similar variant of MoCo for open-domain question answering.

Semantic textual similarity is a long-standing NLP task.Early approaches В (Seco etВ al., 2004; Budanitsky and Hirst, 2001) use lexical resources such as WordNetto measure the similarity of texts.A series of SemEval shared tasks В (Agirre etВ al., 2012, 2014)provide a suite of benchmark datasets that is now widely used for evaluation.Since obtaining large amounts of high-quality STS training data is non-trivial,most STS models are based on weak supervision data,including conversations В (Yang etВ al., 2018),NLI В (Conneau etВ al., 2017; Reimers and Gurevych, 2019),and QA pairs В (Ni etВ al., 2021).

This paper proposes a novel method that aims to solve the easy negativesissue to better align cross-lingual sentence representations.Extensive experiments on multiple cross-lingual and monolingual evaluation datasetsshow the superiority of the resulting representations.For future work,we would like to explore other contrastive learning methods В (Grill etВ al., 2020; Xiong etВ al., 2020),and experiment with more downstream tasksincluding paraphrase mining,text clustering,and bilingual lexicon induction etc.
